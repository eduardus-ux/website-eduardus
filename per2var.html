Pengertian

Persamaan linear dua variabel adalah persamaan yang memiliki bentuk umum: Ax + By = C di mana:

  • x dan y adalah variabel.
  • A, B, dan C adalah bilangan real.
  • Persamaan ini menggambarkan garis lurus pada bidang koordinat.

Bentuk Umum

Contoh bentuk persamaan linear dua variabel:

2x + 3y = 6

Grafik Persamaan Linear

Untuk menggambar grafik persamaan linear, kita perlu menentukan titik potong dengan sumbu x dan y:

  • Titik potong sumbu x: Set y = 0 dan cari x.
  • Titik potong sumbu y: Set x = 0 dan cari y.

Contoh

Untuk persamaan 2x + 3y = 6:

  • Jika y = 0: 2x = 6 → x = 3 (Titik potong x: (3, 0))
  • Jika x = 0: 3y = 6 → y = 2 (Titik potong y: (0, 2))

Menyelesaikan Persamaan Linear Dua Variabel

Untuk menyelesaikan sistem persamaan linear dua variabel, kita bisa menggunakan metode:

  • Metode substitusi
  • Metode eliminasi
  • Metode grafik
